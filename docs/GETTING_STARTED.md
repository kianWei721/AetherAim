# AetherAim — 快速体验指南

## 这是什么

AetherAim 是一个为手部运动障碍玩家设计的 FPS 游戏辅助工具。

**它解决什么问题：**
- 帕金森 / 特发性震颤 → 瞄准时准星高频抖动
- 小脑性共济失调 → 快速甩枪时"刹不住车"（过冲）
- 肌肉疲劳 → 长时间游戏后瞄准精度持续下降
- 反应延迟 → 比别人慢半拍

**它不是什么：**
- ❌ 不是自瞄 / 锁头 / 自动瞄准
- ❌ 不读取游戏内存
- ❌ 不分析游戏画面
- ✅ 它只是一个**智能鼠标滤波器**

类比：AetherAim 之于鼠标，就像降噪耳机之于声音——滤掉噪声（手抖），保留信号（你的瞄准意图）。

    │ 桌面使用      │──────────────│ 方案 A: 桌面软件   │

## 方案 A: 桌面软件 (3 分钟启动)

### 1. 下载

从 [Releases](https://github.com/你的用户名/AetherAim/releases) 下载最新版 `AetherAim.zip`，解压到任意目录。

### 2. 启动

```
右键 AetherAim.exe → "以管理员身份运行"
```

必须管理员权限（WH_MOUSE_LL 全局钩子要求）。

### 3. 校准

点击 **Profile** 标签 → 滚动到底部 → **Start Calibration**。

```
4 个测试，约 2 分钟：
┌─────────────┬──────────┬──────────────────────┐
│ 测试         │ 时长      │ 测量什么               │
├─────────────┼──────────┼──────────────────────┤
│ Steady Aim  │ 5 秒      │ 手抖频率 + 幅度         │
│ Flick Test  │ 8 次点击  │ 甩枪时是否会"甩过头"     │
│ Tracking    │ 20 秒     │ 跟枪准确度 + 疲劳曲线    │
│ Reaction    │ 6 次点击  │ 纯反应速度             │
└─────────────┴──────────┴──────────────────────┘
```

完成后点 **Apply to Profile** → 滤波参数自动调好。

### 4. 日常使用

```
NumLock 或 F8  →  开关滤波

状态栏:
  ● FILTER ACTIVE (绿色)  →  正在滤波
  ○ FILTER PAUSED (红色)  →  直通, 不滤波
```

### 5. 如果不满意

| 感觉 | Settings 标签里调什么 |
|------|---------------------|
| 还是抖 | 减小 **Min Cutoff** (往左拉) |
| 甩枪有延迟 | 增大 **Beta** (往右拉) |
| 光标"不跟手" | 减小 Beta, 增大 Min Cutoff |
| 只想轻度平滑 | 切换到 **EMA** 滤波器, Alpha=0.7 |

每个滑块右侧有 `(?)` 图标，悬停看说明。

## 方案 B: Raw Input 注入 (CS2, Overwatch 2 等)

### 前提

- 已安装方案 A 的桌面软件
- 游戏没有内核级反作弊
- 接受使用风险（CS2 VAC 历史上容忍受信辅助工具, 但不保证）

### 步骤

1. 先启动 AetherAim（管理员）
2. 启动游戏
3. AetherAim → **Games** 标签 → **Refresh Scan**
4. 找到你的游戏 → **Inject**
5. 状态变绿 `INJECTED` → 开始正常工作
6. 退出游戏前点 **Eject**

## 方案 C: Pico 硬件代理 (全部游戏, 包括三角洲/Valorant)

### 为什么需要这个方案

三角洲行动使用 ACE 内核反作弊，Valorant 使用 Vanguard。这些反作弊在内核层阻止任何软件的 DLL 注入和 API Hook。

Pico 方案把滤波做在**物理硬件**上——你的电脑上不运行任何可疑软件。反作弊系统看到的只是一个标准 USB 鼠标。

### 你需要买什么 (¥40)

| 商品 | 价格 | 淘宝搜索 |
|------|------|---------|
| Raspberry Pi Pico 原装 | ¥25 | `树莓派 Pico 原装` |
| Micro-USB 母座转接板 | ¥3 | `micro USB 母座 转接板` |
| 母对母杜邦线 ×4 | ¥2 | `杜邦线 母对母` |
| Micro-USB 数据线 | ¥5 | `micro USB 数据线` |

### 组装 (30 秒, 无需焊接)

```
杜邦线直接插:

  Pico GPIO 0 ──── 母座 D+    (绿线)
  Pico GPIO 1 ──── 母座 D-    (白线)
  Pico Pin 36  ──── 母座 VBUS  (红线)
  Pico Pin 38  ──── 母座 GND   (黑线)
```

### 烧录固件

从 [Releases](https://github.com/你的用户名/AetherAim/releases) 下载 `aether_pico.uf2`:

1. 按住 Pico 上的白色 BOOTSEL 按钮
2. USB 线插电脑
3. 松手 → 出现 RPI-RP2 盘符
4. 把 `aether_pico.uf2` 拖进去
5. 完成

### 使用

```
鼠标 ──USB──► 母座 ──► Pico ──USB──► 电脑

Pico LED:
  慢闪   = 等待鼠标
  快双闪 = 工作中
```

完。没有任何软件需要安装，没有任何进程运行，它就是一根智能 USB 线。
