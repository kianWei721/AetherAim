## 接线指南 (30 秒, 零焊接)

```
                       Pico 正面
              ┌────────────────────────┐
              │  ○ ○ ○ ○ ○ ○ ○ ○ ...  │
              │  GP0 GP1 GND ...       │
              │                        │
              │  [Micro-USB] ←────── 连电脑
              │                        │
              │  Pin36(3V3) Pin38(GND) │
              │  ○ ○ ○ ○ ○ ○ ○ ○ ...  │
              └────────────────────────┘

杜邦线 (母对母) — 直接插, 无需焊接:

  Pico Pin         杜邦线颜色        Micro-USB 母座排针
  ─────────        ────────          ────────────────
  GPIO 0   ──────── 绿 ───────────── 母座 D+
  GPIO 1   ──────── 白 ───────────── 母座 D-
  Pin 36   ──────── 红 ───────────── 母座 VBUS
  Pin 38   ──────── 黑 ───────────── 母座 GND
```

> Micro-USB 母座转接板背面通常印有 VCC/D-/D+/GND/ID 标识。
> D+ 接 Pico GPIO 0，D- 接 GPIO 1。

### 实体图 (文字版)

```
  ┌─────────杜邦线─────────┐
  │                        │
┌─┴──────────┐    ┌────────┴──────────┐
│  Pico      │    │  Micro-USB 母座    │
│  [USB口]───┼────┤  连电脑            │  ←── 鼠标插这里
│            │    │                    │
│  GP0 ○─────┼─绿─┤ D+                 │

└────────────┘    └───────────────────┘
```

## 烧录固件 (60 秒)

### 方法 A: 下载预编译固件 (推荐)

1. 打开 GitHub Releases 页面: `https://github.com/<你的仓库>/releases`
2. 下载 `aether_pico.uf2`
3. **按住 Pico 上的白色 BOOTSEL 按钮**
4. USB 线连接 Pico 到电脑
5. 松开按钮 → 出现 `RPI-RP2` 盘符
6. 把 `aether_pico.uf2` **拖进** RPI-RP2 盘符
7. 完成！Pico 自动重启

### 方法 B: 自己编译 (如果有 Pico SDK)

```bash
git clone https://github.com/raspberrypi/pico-sdk.git
export PICO_SDK_PATH=$(pwd)/pico-sdk

cd pico_firmware
mkdir -p lib
git clone https://github.com/sekigon-gonnoc/Pico-PIO-USB.git lib/pico-pio-usb

mkdir build && cd build
cmake .. && make -j4
# 产物: build/aether_pico.uf2
```

## 首次使用

1. Pico Micro-USB → 电脑
2. 鼠标 → Micro-USB 母座
3. 电脑识别到 "新 USB HID 鼠标"
4. Pico 上的 LED:
   - **慢闪** = 等待鼠标连接
   - **快双闪** = 正常工作
5. 开游戏，正常使用

### 实时调参 (串口控制台)

电脑通过 Pico 的 USB 串口 (115200 baud) 发送命令:

```bash
# 用串口工具连接 Pico 的 COM 口 (Arduino IDE 串口监视器 / PuTTY / minicom)
# 波特率: 115200

help          # 显示所有命令
status        # 查看当前参数
fc 0.8        # 更平滑 (降低截止频率)
beta 30       # flick 延迟更小
dz 2          # 设置 2px 死区
reset         # 重置滤波器
```

**Windows 串口工具:** 设备管理器 → 端口 → 找到 Pico 的 COM 号 → PuTTY/Arduino IDE 连接

## 故障排查

| 问题 | 检查 |
|------|------|
| 电脑不识别设备 | Pico Micro-USB 是数据线吗？换一根线试试 |
| 鼠标不动 | 检查杜邦线方向，D+/D- 是否接反 |
| 鼠标光标乱跳 | D+/D- 接触不良，重插杜邦线 |
| LED 不亮 | 固件是否烧录成功？重新拖 uf2 文件 |
| 串口不显示 | 波特率设为 115200，检查 COM 口号 |
